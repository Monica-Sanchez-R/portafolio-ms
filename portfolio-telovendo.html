Esta página web es una representación inicial de una tienda en línea dedicada a la venta de productos productos del cuidado de pelo. Fue creada utilizando tecnologías web estándar como HTML, CSS y JavaScript y aunque no es funcional en términos de transacciones de compra, demuestra el diseño y la estructura básica de la tienda.
